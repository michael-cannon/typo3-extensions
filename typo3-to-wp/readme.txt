== Description ==
This plugin will import TYPO3 tt_news and tx_comments into WordPress posts and comments. Attachments and media generally included as well.

**Features**

* Import TYPO3 tt_news categories as categories or tags
* Import tt_news entries as posts
* Import tt_news keyword and description meta tags for Thesis
* Import tx_comments as comments for tt_news entries
* Import tt_news images and files as post attachments

[Suggest new features and improvements here](http://peimic.com/contact-us)

**Basic Usage**

After installation, visit the Tools > TYPO3 -> WP Importer

== Installation ==

To do a new installation of the plugin, please follow these steps

1. Download the typo3-to-wp.zip file to your computer.
1. Unzip the file 
1. Upload `typo3-to-wp` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

To upgrade your installation

1. Deactivate the plugin
1. Retrieve and upload the new files (do steps 1. - 3. from "new installation" instructions)
1. Reactivate the plugin. Your settings will be retained from the previous version.
