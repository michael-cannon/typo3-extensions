Install the Extension and configure it with the flexforms!

fields of the flexforms structure
1. url -> you need the complete path to the rss
2. maximum items -> enter the numbers of items
3. objekt divider -> you can add some html tags here like <br /><hr> and so on..
4. description divider -> you can add some html tags here like <br /><hr> and so on..
5. text for submit button -> if the rss file get a form - you can add some text here
6. link target -> select the target for the link
7. display channel description -> boolean
8. display object description -> boolean
9. display image -> boolean
10. display form text -> boolean
11. display feed title -> boolean
12. display link -> boolean
13. display first line -> boolean
14. enable html entities -> boolean
