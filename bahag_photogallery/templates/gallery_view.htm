Gallery View Template

Marker Description:

GALLERY_IMAGE_COUNT - Number of images in current gallery
IMAGE_SRC - Path to gallery image
IMAGE_SOURCE - Source/Provider of gallery image
IMAGE_COMMENT - Comment for gallery image
IMAGE_RESOLUTION - Resolution of gallery image
IMAGE_FORMAT - Format of image like JPEG, GIF, PNG
IMAGE_SIZE - Size of image in KB
IMAGE_INFO_LINK - Link to info page of the gallery image
NORMAL_DOWNLOAD - Download link for gallery image
ZIPPED_DOWNLOAD - Download link for compressed (zip) gallery image
